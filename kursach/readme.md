# Кредитный скоринг: модели машинного обучения для сегментации клиентов

## Суть работы

Построение и сравнительный анализ моделей машинного обучения для многоклассовой классификации кредитного скоринга. На основе данных о клиентах модели автоматически распределяют заёмщиков по трём категориям кредитного рейтинга: **Poor** (плохой), **Standard** (стандартный), **Good** (хороший).

## Данные

- **Источник:** Kaggle — Credit Score Classification
- **Объём:** 100 000 записей, 28 признаков
- **Целевая переменная:** Credit_Score (3 класса)

## Предобработка

- Удалены неинформативные признаки (ID, Name, SSN, Customer_ID)
- Текстовые признаки преобразованы в числовые (Credit_History_Age → месяцы, Num_of_Delayed_Payment → очистка от мусора)
- Обработаны пропуски и выбросы
- Категориальные признаки преобразованы one-hot-encoding
- Для Type_of_Loan (6229 уникальных значений) создан признак Num_Loan_Types

**Результат:** 87 976 строк, 49 признаков, без пропусков.

## Модели и результаты

| Модель | Accuracy | Macro F1 |
|--------|----------|----------|
| Логистическая регрессия | 0.65 | 0.63 |
| LinearSVC | 0.65 | 0.63 |
| **Случайный лес** | **0.78** | **0.76** |
| XGBoost | 0.72 | 0.70 |
| CatBoost | 0.71 | 0.68 |

**Лучшая модель:** Случайный лес

## Ключевые признаки

Наибольший вклад в предсказание вносят:
1. Непогашенный долг (Outstanding_Debt)
2. Процентная ставка (Interest_Rate)
3. Просрочка платежа (Delay_from_due_date)
4. Возраст кредитной истории (Credit_History_Age)
5. Количество кредитных запросов (Num_Credit_Inquiries)

## Сегментация клиентов

| Сегмент | Риск | Рекомендации |
|---------|------|--------------|
| Good | Низкий | Лучшие условия кредитования |
| Standard | Средний | Стандартные условия |
| Poor | Высокий | Отказ или кредит под залог |

## Вывод

Разработан полный пайплайн предобработки данных и построения моделей кредитного скоринга. Лучшая модель (случайный лес) достигла Accuracy 0.78 и Macro F1 0.76. Результаты могут быть использованы для автоматизации принятия решений при кредитовании.
